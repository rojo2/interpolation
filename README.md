# `@rojo2/interpolation`

Utility functions to use `linear`, `quadratic` and `cubic` interpolation.

```javascript
import { linear, quadratic, cubic } from '@rojo2/interpolation'

linear(0.5, -100, 100)
quadratic(0.5, -100, 0, 100)
cubic(0.5, -100, 0, 0, 100)
```
